# Edhen POS Frontend

Sistema de Punto de Venta (POS) para gestionar ventas, clientes, productos y reportes.

## Características

✅ Dashboard inicial  
✅ Gestión de Clientes  
✅ Gestión de Productos  
✅ Registro de Ventas  
✅ Reportes  
✅ Validaciones mejoradas  

## Scripts Disponibles

### `npm start`
Ejecuta la app en modo desarrollo en [http://localhost:3000](http://localhost:3000)

### `npm test`
Ejecuta los tests

### `npm run build`
Crea una versión optimizada para producción

## Estructura del Proyecto

```
src/
├── components/   - Componentes de React
├── services/     - Servicios de API
├── utils/        - Utilidades (validadores)
└── App.js        - Componente principal
```
